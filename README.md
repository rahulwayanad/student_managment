# Student Management System
Rahul Manayath

## Requirements  - Laravel 9

- PHP -  8
- Node - 18+
- MySql
- Npm - 6+

## Steps

Run following commands

```sh
git clone https://github.com/rahulwayanad/student_managment.git
cd student_managment
composer install
npm install
npm dev run
php artisan migrate
php artisan serve
```
